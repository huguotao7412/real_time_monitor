# README / Operation Manual Design

## Goal

Create a bilingual (EN/ZH) README.md and operation manual covering the full
project: what it does, how to install, how to use, architecture for developers.

## Structure

1. Project Overview (EN → ZH)
2. Features (EN → ZH)
3. System Requirements
4. Installation
5. Quick Start (replay / serial / BP)
6. Operation Manual (UI layout, HR mode usage, BP mode usage, mode switch, data export)
7. CLI Arguments (table)
8. Project Architecture (directory tree + module descriptions)
9. Configuration Reference
10. FAQ

## Format

Each section: English first, then Chinese below. Code blocks and CLI commands
use English. UI component names kept in English with Chinese explanation.

## Files

- Create: `README.md` at project root

双模式架构：受试者模式 vs 研究模式应该如何切换？

这个决定影响整个 UI 结构，请选择最合适的方式

A

单窗口 + 模式切换开关

一个窗口，右上角放置醒目的模式开关。受试者模式隐藏坐标轴、网格线、调试面板。研究模式展开侧边栏显示 DSP 内部数据。切换平滑，无需管理多窗口。

适合：单屏幕场景，演示时快速切换

B

两个独立窗口

受试者模式打开一个极简全屏窗口（大字 BPM、柔和波形）。研究模式是另一个独立窗口，包含所有 DSP 面板。完全分离，但需要管理两个窗口。

适合：双屏幕场景（受试者看一屏，研究人员看另一屏）

C

标签页布局（受试者标签 / 研究标签）

单窗口顶部两个标签页。受试者标签是干净极简视图。研究标签包含所有高级面板、趋势图和调试数据。熟悉的标签模式，但需要显式切换。

适合：有序工作流，清晰的心智模型分离

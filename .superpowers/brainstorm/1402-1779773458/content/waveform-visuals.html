波形"生命感"视觉设计

当前：裸绿色折线。目标：让界面在"呼吸"。

1. 呼吸波形填充

WaveWidget 使用 pg.FillBetweenItem 或 setFillLevel(0)，在曲线下方填充半透明渐变色。
颜色：从底部 深绿 过渡到顶部 亮绿，模拟肺容量变化的"体积感"。

受试者模式：隐藏 X/Y 坐标轴、隐藏网格。研究模式：保留坐标轴和网格。

2. 呼吸律动圆圈 (BreathingCircle)

BPM 数字旁一个空心圆，用 breath_bpm 频率驱动缩放动画。

A

正弦缩放圆

圆的半径随呼吸相位做正弦波动的缩放，幅度 20%。呼气时缩小，吸气时放大。
直接绑定当前相位信号 (displacement) 的实时值做映射，不依赖 BPM 数字。

最直观、与真实呼吸同步、实现简单

B

固定频率呼吸花瓣 (Apple Watch 风格)

多层花瓣以 breath_bpm 频率周期性开合。BPM 变化时花瓣速度随之变化。
使用 QPropertyAnimation 驱动旋转/缩放。

精致美观、但有 ~2s 的 BPM 更新延迟、与瞬时相位不同步

3. 心率脉动图标

心率数字旁显示一个心形图标（用 QPainterPath 绘制贝塞尔曲线），以 heart_bpm 频率缩放跳动。
颜色阈值：60-100 绿色 → 50-60 或 100-110 橙色 → <50 或 >110 红色

4. SQI 信号质量灯

右上角三格信号图标。综合 breath_ratio + phase_range + heart_prominence 计算 SQI 等级：

等级颜色条件BPM 显示
优● 绿色 breath_ratio > 0.15 且 phase_range > 0.01 正常显示
中● 黄色 breath_ratio 0.05-0.15 或 phase_range 0.005-0.01 正常显示
差● 红色 breath_ratio < 0.05 或 phase_range < 0.005 半透明/灰显

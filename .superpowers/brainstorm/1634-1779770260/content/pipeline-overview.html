Pipeline Transformation: Current vs. Proposed

RS6240 雷达生命体征算法 — MATLAB 高级方案向现有 Python Pipeline 的移植

Current Pipeline (现有链路)

dsp_pipeline/pipeline.py — _process_frame()
1. CFAR → 锁定最佳 Range Bin (range_bin.py)
2. 相位提取 → 4 RX 天线 取均值 → arctan2 (phase.py)
3. 预处理 → 解缠 + 去直流 + 一阶差分 (filters.py)
4. 滤波 → SOS Butterworth 带通 (呼吸 0.1-0.8, 心率 0.8-2.5) (filters.py)
5. 谐波抑制 → 频域 FFT 谐波软衰减 (harmonic_mask.py)
6. BPM 估计 → 时域寻峰(呼吸) + FFT(心率) + Kalman (fft_bpm.py)

Proposed Pipeline (MATLAB 移植方案)

新链路 — 三步替换, 每步独立验证
1. CFAR → ◎ 保持不变 (range_bin.py)
2. LCMV 波束成形 → MVDR 角度估计 → LCMV 加权 → 相位提取 (NEW: lcmv_beamformer.py)
3. EMD 谐波对消 → EMD 分解 → 呼吸基频 → 最小二乘对消 (NEW: emd_cleaner.py)
4. WPD 分离 → db6 提取呼吸 · sym8 提取心跳 (NEW: wpd_filter.py)
5. STFT + Kalman → STFT 时频脊线 → Kalman 平滑 (修改: fft_bpm.py)

实施策略: 三步渐进替换

Step 替换模块 新增文件 风险
Step 1 SOS滤波 → EMD+WPD emd_cleaner.py, wpd_filter.py ⚠ 中 — EMD 计算量大
Step 2 均值相位 → LCMV 波束成形 lcmv_beamformer.py ⚠ 中 — 需要验证角度估计
Step 3 FFT寻峰 → STFT+Kalaman 修改 fft_bpm.py ✓ 低 — 复用现有 Kalman
